## About This Add-on

This add-on provides a modern Home Assistant integration with a **FastAPI backend**
and **React frontend** served via the HA Ingress interface.

### Features

- **REST API** — FastAPI endpoints for Home Assistant interaction
- **WebSocket** — Real-time bidirectional communication
- **Ingress UI** — Embedded React dashboard directly in the HA sidebar
- **Entity Control** — Query and control Home Assistant entities
- **Multi-Stage Docker Build** — Optimized production image
